<h1 align="center">GraphQL Employee Portal</h1>

<div align="center">

Employee portal built with GraphQL using Apollo Server and React with Apollo Client.

</div>

## Running Client

Install necessary dependencies:

```
  npm install
```

Start application:

```
  npm run start
```

## Running Server

Install necessary dependencies:

```
  npm install
```

Start application:

```
  npm run start
```
